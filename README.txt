==============================
TXT 转 Excel 批量转换器
==============================

功能简介
----------
1. 将文件夹中的 TXT 文件批量转换为 Excel (.xlsx) 文件。
2. 支持指定要提取的列（列号从 1 开始）。
3. 自动检测文件编码（UTF-8、GBK、ANSI、UTF-16）。
4. 自动识别分隔符（逗号、制表符、空格）。
5. 第一行保持原始文本格式。
6. 第一列保持文本格式，其余列转为数字。
7. 显示处理进度条和状态信息。

依赖环境
----------
- Python 3.7+
- 依赖库：pandas, tqdm, openpyxl

安装依赖示例：
    pip install pandas tqdm openpyxl

使用方法
----------
1. 将要转换的 TXT 文件放在同一个文件夹中。
2. 修改脚本 txt2excel.py 中 folder_path：
    folder_path = r"D:\data\txt_files"
3. 运行脚本：
    python txt2excel.py
4. 输入要提取的列号（以半角逗号分隔）：
    例如：1,3,5
5. 程序会生成 Excel 文件在同一目录下，与 TXT 文件同名。

示例
----------
TXT 内容：
    ID  A  B  C  D
    001 10 20 30 40
    002 15 25 35 45

输入列号：1,3,4

生成 Excel：
    ID  B  C
    ID  B  C
    001 20 30
    002 25 35

注意事项
----------
1. 列号从 1 开始计数，不能超过 TXT 文件的总列数。
2. 文件夹路径建议使用绝对路径，并加 r"..." 前缀。
3. 非数字列转换失败会显示空白（NaN）。
4. 大文件处理可能需要一些时间，请耐心等待。
